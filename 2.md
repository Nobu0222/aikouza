## XORを作る

![gate](gate.png)

cmd起動　python と入力

```python
def AND(a, b):
  if a == 1 and b == 1:
    return 1
  else:
    return 0
```

```python
def OR(a, b):
  if a == 0 and b == 0:
    return 0
  else:
    return 1
```

```python
def NOT(a):
  if a == 1:
    return 0
  else:
    return 1:
```

![xor](XOR.png)

```python
def XOR(a, b):
  c = AND(a, b)
  c = NOT(c)
  d = OR(a, b)
  return AND(c, d)
```

```
>>>XOR(0, 0)
0
>>>XOR(1, 0)
1
>>>XOR(0, 1)
1
>>>XOR(1, 1)
0
```
